{% trans "Recommandations importantes" %}

{% blocktrans %} Ces quelques recommandations vous permettront de proposer un événement qui sera validé plus rapidement, et dont la lisibilité sera meilleure pour les utilisateurs de l'Agenda. {% endblocktrans %}

  • {% blocktrans %} Le texte des événements doit être rédigé en français, en évitant les fautes de grammaire et d'orthographe. {% endblocktrans %}
  • {% blocktrans %} Donnez une description rédigée de l'événement, avec des phrases, plutôt qu'un style télégraphique. {% endblocktrans %}
  • {% blocktrans %} Commencez la description de votre événement par une phrase reprenant les informations principales, telle que : « La maison de Quartier des Dervallières organise un spectacle de danse Jazz le mardi 19 juillet à 20h. » {% endblocktrans %}
  • {% blocktrans %} Pensez que le lecteur peut ne pas connaître votre lieu, le sujet dont il sera question, etc., donc donnez à chaque fois toutes les informations nécessaires. Rappelez le principe de l'événement, le public visé, le lieu exact, la date, l'heure, même si il est régulier (repas, rencontre régulière, etc.) et que vous avez déjà soumis un événement du même type dans l'Agenda. {% endblocktrans %}
  • {% blocktrans %} Égayez au maximum la description de votre événement: lien vers un site de présentation, vers les organismes partenaires, etc. Pour cela, utilisez les balises HTML pour mettre en gras, en italique, faire des paragraphes et des liens. {% endblocktrans %}
  • {% blocktrans %} Ne mettez pas l'intégralité des mots en majuscule, que ce soit dans le titre, le nom de la ville ou la description. {% endblocktrans %}
  • {% blocktrans %} Donnez l'adresse directe d'un site Web donnant plus d'informations sur l'événement. Faites un lien qui pointe directement vers l'événement, et non vers une page d'accueil. {% endblocktrans %}
  • {% blocktrans %} La description de l'événement ne doit pas ressembler à une publicité éhontée pour votre projet. L'évènement doit être organisé, ou co-organisé par l'Accoord. {% endblocktrans %}

{% blocktrans %} L'équipe de modération se réserve le droit de modifier la description de l'événement pour la rendre plus complète, plus lisible ou plus attrayante. {% endblocktrans %}
